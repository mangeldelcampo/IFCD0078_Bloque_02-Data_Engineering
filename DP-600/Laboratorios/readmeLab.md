
# Índice de Laboratorios

| # | Laboratorio | Descripción |
| :--- | :--- | :--- |
| 1 | [Introducción a la inteligencia en tiempo real en Microsoft Fabric](./1.Get_started_with_Real_Time_Intelligence_in_Microsoft_Fabric/readme1.md) | En este laboratorio, aprenderá a trabajar con flujos de datos en tiempo real en Microsoft Fabric, ingiriendo datos del mercado de valores mediante flujos de eventos, consultándolos con KQL y visualizándolos en paneles en tiempo real. También configurará alertas con Activator para responder a eventos en tiempo real. |
| 2 | [Diseño e implementación de un modelo semántico](./Laboratorios/2.Design_and_implement_a_dimensional_model/readme2.md) | En este laboratorio, diseñará e implementará un modelo dimensional de esquema en estrella en un Fabric Warehouse, creando tablas de hechos y dimensiones con relaciones de clave externa. Ejecutará consultas analíticas para agregar datos de ventas en múltiples dimensiones e implementará patrones de dimensión de cambio lento (SCD) de tipo 1 y tipo 2 para manejar datos que cambian con el tiempo. |
| 3 | [Diseñar modelos semánticos escalables](./3.Design_scalable_semantic_models/readme3.md) | En este laboratorio, diseñará un modelo semántico para escalabilidad en el servicio Microsoft Fabric. Se conectará a los datos de Lakehouse mediante Direct Lake, creará relaciones de esquema en estrella, creará un grupo de cálculo para inteligencia temporal y configurará ajustes que admitan grandes conjuntos de datos y consumo simultáneo. |
| 4 | [Optimizar el rendimiento de los modelos semánticos](./4.Optimize_semantic_model_performance/readme4.md) | Utilice el Analizador de rendimiento de Power BI Desktop para diagnosticar el rendimiento visual de los informes, identificar patrones DAX costosos, aplicar optimizaciones y verificar las mejoras. |
| 5 | [Garantizar la seguridad de los modelos semánticos](./5.Enforce_semantic_model_security/readme5.md) | Importe una tabla de asignación de seguridad, configure la propagación del filtro de relaciones e implemente la seguridad estática y dinámica a nivel de fila en un modelo semántico de Power BI. |
| 6 | [Manage the semantic model lifecycle](./6.NOMBRE_LABORATORIO/readme6.md) | Validar un modelo semántico con SemPy en un cuaderno de Fabric, crear una canalización de implementación para promover el contenido entre las distintas etapas y verificar el contenido implementado en un espacio de trabajo de producción.|
| 7 | [Prepare a semantic model for AI](./7.NOMBRE_LABORATORIO/readme7.md) | Configure las funciones de preparación para IA en Power BI Desktop, agregue sinónimos mediante modelado lingüístico, publíquelo en un espacio de trabajo de Fabric, realice pruebas con Copilot y los diagnósticos de HCAAT, y marque el modelo como aprobado para Copilot. |
| 8 | [Create an ontology with Fabric IQ](./8.NOMBRE_LABORATORIO/readme8.md) | En este laboratorio, crearás manualmente una ontología completa de Fabric IQ mediante la creación de tipos de entidades con propiedades y claves, la definición de relaciones y la vinculación de datos de tablas de Lakehouse y flujos de Eventhouse. Trabajarás de forma práctica con datos estáticos y de series temporales para modelar un escenario de atención médica que incluye hospitales, departamentos, pacientes y equipos de monitorización de signos vitales. |
| 9 | [Build an ontology from a semantic model in Fabric IQ](./9.NOMBRE_LABORATORIO/readme9.md) | En este laboratorio, generará una ontología de Fabric IQ a partir de un modelo semántico de Power BI utilizando datos de atención médica, configurará tipos de entidades y relaciones con enlaces de datos, y combinará datos estáticos de un lakehouse con datos de series temporales de un eventhouse. |
